
  # VidFlow Studio Design

  This is a code bundle for VidFlow Studio Design. The original project is available at https://www.figma.com/design/pUW3Jwub3Rx796QbNokrWJ/VidFlow-Studio-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Integration tests + Playwright MCP

  - E2E tests: `npm run test:e2e`
  - Playwright MCP server: `npm run mcp:playwright`
  - Walkthrough: `INTEGRATION_TESTS.md`
